# 🔒 C++ YARA-Based PDF Malware Scanner

High-performance PDF malware detection using YARA rules and C++17.

## 📦 Dependencies

```bash
# Ubuntu/Debian
sudo apt-get install cmake g++ libyara-dev libpodofo-dev libssl-dev nlohmann-json3-dev

# Fedora/RHEL
sudo dnf install cmake gcc-c++ yara-devel podofo-devel openssl-devel nlohmann-json-devel

# macOS
brew install cmake yara podofo openssl nlohmann-json
